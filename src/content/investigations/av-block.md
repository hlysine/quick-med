# Atrioventricular Conduction Block

## First degree

![First degree AV block](/assets/av-block-1st-degree.png)

* Prolonged PR interval >210ms
* No symptoms

## Second degree Mobitz type 1

![Mobitz I strip](/assets/av-block-mobitz-1-strip.png)

![Mobitz I ladder diagram](/assets/av-block-mobitz-1-ladder.png)

* PR interval increases progressively until QRS blocked
* QRS pause not a multiple of normal RR interval
* No symptoms

## Second degree Mobitz type 2

![Mobitz II](/assets/av-block-mobitz-2.png)

* Constant PR interval but intermittent QRS block
* QRS pause is a multiple of normal RR interval
* Likely to progress to third degree heart block
* Likely to have wide QRS complexes
* Requires permanent pacemaker

## Second degree high grade block

![High grade AV block](/assets/av-block-high-grade.png)

* Mobitz type 2 with high ratio of QRS block (P:QRS > 3:1)
* Requires evidence of intermittent conduction or constant PR interval
    * Intermittent narrow QRS complex

## Third degree complete heart block

![Complete heart block](/assets/av-block-complete.png)

* Atrium and ventricle beat independently
* Atrial rate faster than ventricular rate
* If escape beat generated by junction - block at AV node
    * Narrow QRS
* If escape beat generated by ventricles - block at His bundle
    * Wide QRS
