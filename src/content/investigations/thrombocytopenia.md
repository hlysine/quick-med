# Thrombocytopenia

* Most of the time has **known cause** (e.g. haematology patients)
* If sudden onset, **consider pseudo-thrombocytopenia** (due to platelet clumping) **and drug-induced**
    * Recheck CBC (in citrated blood)
    * Peripheral blood smear (PBS)
    * **Recent use of heparin, antibiotics, anti-epileptics**
* **Look out for MAHA (e.g. DIC/ TTP/ HUS)**
    * Check PBS (for schistocytes), retic, LDH, haptoglobin, unconjugated bilirubin, clotting, RFT
* **Unless life-threatening bleeding, transfusion is NOT usually indicated in ITP, SLE, TTP, HUS, DIC**

> If pancytopenia + DIC, must think APL! Urgent find haematopathologist for PBS interpretation -> **give ATRA immediately**

> Ask for any **neurological symptoms** and any **recent trauma/ head injury** - no matter HOW minor it is, with low platelet there is still risk of ICH!
> * Chart GCS + pupil size
> * **Urgent CT brain**
> * Neuro-obs

### Indications for transfusion

> Each unit of platelet raises Plt count by ~ 7-10 x 10^9/L

* Platelets < 10 x 10^9/L
* Platelets < 20 x 10^9/L with FEVER
* Platelets < 50 x 10^9/L with bleeding or before invasive procedures
* Platelets < 100 x 10^9/L before eye or neurosurgery
* Suspected platelet dysfunction with active bleeding or before invasive procedures

**Common transfusion order: 4 units or 6 units platelets full rate**

> Transfusion of components (platelet, FFP etc.) does not need T&S, as long as this patient has had a T&S done in this hospital in their lifetime!
