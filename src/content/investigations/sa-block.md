# Sinoatrial Conduction Block

## First degree SA block

Completely invisible on ECG.

## Second degree Mobitz type 1

![SA Mobitz I strip](/assets/sa-block-mobitz-1-strip.png)

![SA Mobitz I ladder diagram](/assets/sa-block-mobitz-1-ladder.png)

* RR interval progressively shortens until a beat is skipped
* No symptoms

## Sinoatrial arrest / pause

![Sinus arrest strip](/assets/sa-arrest-strip.png)

![Sinus arrest ladder diagram](/assets/sa-arrest-ladder.png)

* Normal constant RR interval
* Pause interval is not a multiple of normal RR interval
* Syncope if pause is too long

## Second degree Mobitz type 2

![SA Mobitz II strip](/assets/sa-block-mobitz-2-strip.png)

![SA Mobitz II ladder diagram](/assets/sa-block-mobitz-2-ladder.png)

* Constant RR interval but intermittent SA block
* Long RR interval is a multiple of normal RR interval
* No symptoms
