# COVID (SARS-CoV2, nCoV)

* Droplet precautions; no need isolation anymore
* Only need to report NDORS if desat > 4L

## Antivirals and DVT prophylaxis for COVID

### For mild symptomatic high-risk patients within 5 days of symptom onset
1. **Paxlovid (preferred option)**
    * If eGFR >60: PO Paxlovid 300mg/100mg (300mg nirmatrelvir/ 100mg ritonavir) Q12H x 5 days
    * If eGFR 30-60: PO Paxlovid 150mg/100mg Q12H x 5 days
    * Drug-drug interaction+++ (statin being most common)
    * **Stop statin during course of Paxlovid**
2. **Molnupiravir (less effective but less drug interactions)**
    * If contraindicated for Paxlovid
    * PO 800mg Q12H

> Use **Lexicomp** (in UpToDate) to check for drug interactions! UpToDate can be accessed in CMS via e-Knowledge Gateway (eKG)

### If desaturation (SpO2 < 94% on RA) or new CXR infiltrates
1. **Remdesivir**
    * C/I if ↑ ALT >= 5x ULN or eGFR <30
    * IV remdesivir 200mg stat then 100mg Q24H x 4-9 days (total 5-10 days)
2. **Dexamethasone**
    * IV Dexamethasone 6mg stat then Q24H x 9 days (total 10 days)
    * Alternative: hydrocortisone 100mg IV stat then 50mg Q8H or prednisolone 40mg PO then daily
    * Check HBsAg, monitor H'stix BD, add PO Pantoloc 40mg daily during course of steroid
3. **Adjunctive immunomodulator** (baricitinib / tocilizumab)
    * Consult MED/ID for immunomodulator if worsening/no improvement and ↑ CRP
4. **Thromboembolism prevention** if not already on anticoagulation
    * Prophylactic dose of LMWH (Clexane 1mg/kg BW Q24H) **if no C/I**
        * Outpatient: only need if prior VTE, recent surgery or trauma, or cast immobilisation
        * Inpatient: newest trials suggest therapeutic dose (Clexane 1mg/kg BW Q12H) is superior
    * TED stocking
    * Early mobilisation if possible
    * Off before d/c if no indication for long-term use
