Suspect if any discomfort during transfusion.

## Causes

- **Haemolytic reaction**: ABO incompatibility, alloantibodies
- **Febrile non-haemolytic reaction (FNHTR)**: cytokines from donor leukocytes — most common
- **Allergic/anaphylactic**: donor plasma proteins
- **TACO** (transfusion-associated circulatory overload): fluid overload in susceptible patients
- **TRALI** (transfusion-related acute lung injury): donor antibodies against recipient leukocytes
- **Septic reaction**: bacterial contamination of blood product
- **Delayed haemolytic reaction**: anamnestic antibody response days post-transfusion

## Complications

- Acute haemolysis → haemoglobinuria, AKI, DIC
- TACO → pulmonary oedema, respiratory failure
- TRALI → non-cardiogenic pulmonary oedema
- Anaphylaxis → cardiovascular collapse
- Septic shock

## Assessment

- **Stop transfusion immediately** and disconnect transfusion set from patient
- Vitals (temperature, BP, HR, SpO2, RR)
- Assess for: fever, rigors, urticaria/rash, SOB, chest/back/loin pain, haemoglobinuria (dark urine), hypotension
- Check blood unit label against patient identity and blood type
- Inspect blood unit for signs of contamination (discolouration, clots, haemolysis, gas bubbles)

## Management

### Immediate Steps

1. **Stop transfusion + DISCONNECT transfusion set from patient**
2. Maintain IV access with new IV set and normal saline
3. Notify MO; **report AIRS** (confirm with MO — not mandatory if fever is clearly FNHTR)
4. Check blood unit for blood type match and signs of contamination
5. **Return blood unit + transfusion set to blood bank/microbiology lab** for testing

### Investigations

| Investigation | Purpose |
|---|---|
| Repeat T&S | Confirm compatibility |
| Blood c/st (if fever) | Rule out septic reaction |
| CBC d/c | Haemolysis, thrombocytopaenia |
| L/RFT | AKI (haemolysis), liver involvement |
| Clotting, fibrinogen, D-dimer | DIC |
| Haptoglobin, LDH, reticulocytes | Haemolysis markers |
| DAT (direct antiglobulin test) | Immune-mediated haemolysis |
| Urine x haemoglobin | Haemoglobinuria |

- If SOB/chest discomfort: + **CXR, ECG**, ± ABG (distinguish TACO vs TRALI)

### Treat Underlying Cause

- **Haemolytic reaction**: aggressive IVF, maintain urine output, nephrology consult if AKI; treat DIC if present
- **FNHTR**: antipyretics; may restart transfusion with new unit if no other concern (confirm with MO)
- **Allergic/urticaria**: antihistamine; if anaphylaxis — adrenaline IM, IV hydrocortisone
- **TACO**: diuresis (IV frusemide), elevate HOB, supplemental O2
- **TRALI**: supportive — supplemental O2, possible intubation; avoid diuresis (not fluid overload)
- **Septic reaction**: blood c/st → broad-spectrum antibiotics; ICU consult
