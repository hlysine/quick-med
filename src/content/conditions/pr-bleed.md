# PR Bleed

## Causes
* **Outlet type:** haemorrhoid, rectal polyp, proctitis, rectal ulcer, anal fissure, anal fistula
* **Colon:** polyp, **cancer, diverticulosis**, angiodysplasia, colitis (infective/ inflammatory/ ischaemic/ radiation)
* **Small bowel:** angiodysplasia, Meckel's diverticulum, enteritis (infective/ inflammatory/ ischaemic)
* **Massive upper GI bleed**
* Systemic: antiplatelets/anticoagulants, coagulopathy

> **Pitfall:** red dragon fruit causes "red stool", can see seeds (and it looks pink!)

## Assessment
* Vitals (considered unstable if tachycardia)
* +/- Hemocue (POCT for Hb level)
* Amount of PR bleed
* Inspect napkin/bedpan/toilet bowl: **fresh blood (+/- clot) vs altered blood / melaena**
* Symptoms (abdominal pain/ anal pain/ anaemic symptoms)
    * *Anaemic symptoms, Hb level and rate of bleeding help determine need for transfusion*
* Abdominal exam
* **PR exam and proctoscopy**
    * PR: bleeding stopped vs active; fresh blood (+/- clot) vs altered blood / melaena
    * Proctoscopy: haemorrhoid (internal/ external), rectal ulcer/polyp, vs blood from above
* Background history, previous abdominal imaging/endoscopy

> **Tips for putting adrenaline gauze:** remember to stitch them together, document the number of gauzes and document complete removal!

## Management
* Fluid resuscitation if unstable (consider direct blood transfusion if massive bleeding)
* **If significant PR bleed:**
    * NPO, W/H DM meds during fasting, maintenance IVF
    * Obs Q1H x4 then Q4H if stable
* Chart PR bleed, I/O
* **Urgent CBC, L/RFT, clotting, +/- VBG, lactate +/- T&S;** transfuse to keep Plt > 50, INR < 1.5
* **Urgent AXR**
* **W/H antiplatelets** (consult if DAPT for ACS), **W/H +/- reverse anticoagulation**
* **If rectal source identified (outlet type bleeding):**
    * Slow oozing (from mucosal abrasion): **adrenaline gauze x 24 hours or until it slips out**
    * Spurting: adrenaline gauze + **urgent consult SUR x plication**
* **If unstable / active massive bleeding / significant Hb drop: urgent consult SUR**
    * Localise bleeding source: 1st step is urgent OGD to exclude UGIB
    * If OGD -ve: EOT if unstable; CTA if stable ongoing bleeding; CLN if stopped/mild
* If minor PR bleed with no known cause: routine consult SUR (or MED GI) x LGI workup
