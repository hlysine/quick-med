## Causes

| Location | Examples |
|---|---|
| Outlet | Haemorrhoid, rectal polyp, proctitis, rectal ulcer, anal fissure, anal fistula |
| Colon | Polyp, cancer, diverticulosis, angiodysplasia, colitis (infective/inflammatory/ischaemic/radiation) |
| Small bowel | Angiodysplasia, Meckel's diverticulum, enteritis |
| Upper GI | Massive UGIB (melaena) |
| Systemic | Antiplatelets/anticoagulants, coagulopathy |

## Assessment

- **Vitals** (tachycardia = unstable)
- ± Hemocue
- Amount; **inspect napkin/bedpan:** fresh blood ± clot vs altered blood/melaena
- Symptoms: abdominal pain, anal pain, anaemic symptoms
- Abdominal exam
- **PR + proctoscopy:** bleeding stopped vs active; type; haemorrhoid/rectal ulcer/polyp vs blood from above
- Background + previous imaging/endoscopy

## Management

### Resuscitation

- Fluid resuscitation if unstable (**direct blood transfusion** if massive)

### If Significant PR Bleed

- NPO, W/H DM meds, maintenance IVF
- Obs Q1H ×4, then Q4H if stable
- Chart PR bleed + I/O
- Urgent CBC + L/RFT + clotting ± VBG + lactate ± T&S
- **Transfuse to keep Plt >50, INR <1.5**
- Urgent AXR
- W/H antiplatelets (consult if DAPT); W/H ± reverse anticoagulation

### If Rectal Source Identified (Outlet Type)

| Bleeding | Action |
|---|---|
| Slow oozing | Adrenaline gauze ×24hrs or until slips out |
| Spurting | Adrenaline gauze + urgent consult SUR x plication |

### If Unstable / Active Massive / Significant Hb Drop

- Urgent consult **SUR**

### Localising the Source

1. **Urgent OGD** first to exclude UGIB
2. If OGD negative:
   - Unstable: **EOT** (emergency operative treatment)
   - Stable, ongoing: **CTA**
   - Stopped/mild: **CLN** (colonoscopy)
3. Minor PR bleed, unknown cause: routine consult **SUR/GI** x LGI workup
