## Causes by Pattern

| Pattern | Causes |
|---|---|
| **Hepatitic** (elevated ALT) | Hepatitis A/E/B/C, alcohol, drug-induced / TCM, haemochromatosis, autoimmune hepatitis, HCC |
| **Cholestatic** (elevated ALP) | Cholangitis, biliary obstruction (stones / malignancy / stricture e.g. PSC / PBC) |
| **ALT elevation (non-hepatic)** | Myositis |
| **ALP elevation (non-hepatic)** | Bone conditions, pregnancy |
| **Bilirubin elevation (non-hepatic)** | Haemolysis |

---

## Complications

- **Synthetic dysfunction**: coagulopathy, hypoalbuminaemia
- **Metabolic dysfunction**: hypoglycaemia, hepatic encephalopathy (HE)

---

## Assessment

- History: fever, TOCC, alcohol intake, drugs (including TCM), history of CLD
- Physical exam:
  - Vitals (especially fever) + H'stix + GCS
  - I/O (ensure BO if CLD)
  - Jaundice (tea-coloured urine in urine bag)
  - Stigmata of CLD (if acute-on-chronic)
  - RUQ mass / hepatomegaly
  - Ascites, lymph nodes

---

## Management

### Blood Investigations
- AST + GGT + heat stable ALP + conjugated bilirubin + clotting + NH3 + random glucose
- HBsAg + anti-HCV + anti-HAV IgM + anti-HEV IgM
- Iron profile + serum toxicology + alcohol level
- Paracetamol level (if indicated)
- AFP + CEA (if malignancy suspected)

### Pattern Interpretation
- **AST:ALT >2:1 + GGT** → suggests alcoholic liver disease
- **Heat stable ALP + GGT + conjugated bilirubin** → suggests cholestasis

### Additional Workup
- Urine toxicology (if indicated)

### Drug Management
- Off **TB drugs**
- Off / reduce **Paracetamol** (change to PRN or reduce dose; <2 g/day not associated with acute decompensation)
- Off **statins**

### Specific Situations
- If **fever / sepsis + deranged LFT**: blood culture; urgent USG hepatobiliary system; urgent consult surgery if positive findings
- If **alcoholic**: thiamine supplementation
- If **clotting derangement**: vitamin K

### Monitoring
- Monitor: **LFT + clotting + NH3 + random glucose**
- **Consult MED** if unexplained and persistent
