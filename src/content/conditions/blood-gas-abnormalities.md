## ABG vs VBG

- Look at O2 saturation: if low without history of desaturation, it is likely a **VBG**
- **VBG**: only metabolic parameters are meaningful — **pH / HCO3 / BE**; ignore pO2 / pCO2 / O2 sat
- **Exception**: if VBG pH is low but HCO3 / BE are not low — suspect **respiratory acidosis**; take ABG to confirm, especially if pCO2 is very high

## Respiratory Acidosis (Type 2 Respiratory Failure / Hypoventilation)

### If Weaning ABG
- Put patient back on most recent ventilator settings

### If on Ventilatory Support
- **AECOPD on NIV**: call MO — consider mask leak / circuit leak / over-oxygenation / inadequate IPAP / insufficient time on NIV
  - If already optimised: **failed NIV → needs prompt intubation**
- **Other situations**: increase RR by 2 (up to 20) and IPAP by 2 (up to 20), or increase TV by 50 ml (up to 8 ml/kg IBW); repeat ABG in 1–2 hrs

### If Not on Support
- Call MO to discuss intubation or BiPAP

### Ventilator Settings (Invasive)
| Parameter | Value |
|---|---|
| Mode | SIMV |
| RR | 12–16 |
| Tidal Volume | 6–8 ml/kg IBW (~400–500 ml) |
| FiO2 | Titrate from 1.0 |
| PEEP | 5 |
| I:E ratio | 1:2 (1:3 obstructive; 1:1.5 ARDS) |

- **Sedation**: Midazolam (Dormicum) +/- Morphine 45 mg in 45 ml NS, start 1 ml/hr; withhold if SBP <90

### BiPAP Settings
| Parameter | Value |
|---|---|
| IPAP | 14 |
| EPAP | 4 |
| RR | 14 |
| FiO2 | 0.4 |

- Repeat ABG 1 hr after initiation
- **Do NOT give NaHCO3 for respiratory acidosis**

## Respiratory Alkalosis (Hyperventilation)

- **If on ventilator / BiPAP**: decrease IPAP by 2, TV by 50 ml, or RR by 2; repeat ABG 1 hr later
- **If not on support**: advise patient to stop hyperventilating
- **Pitfall**: in asthma exacerbation, expect hypocapnia (low pCO2); a **normal pCO2** is a warning sign of a severe attack

## Metabolic Acidosis

### Anion Gap Calculation
- **AG = Na − Cl − HCO3** (normal 4–12)
- Alternatively: **AG = Na + K − Cl − HCO3** (normal 8–16)

### High AG Metabolic Acidosis (HAGMA) — Mnemonic: KULT
| Cause | Examples |
|---|---|
| **K**etoacidosis | DKA, alcoholic ketoacidosis, starvation |
| **U**raemia | Renal failure |
| **L**actate | Shock / hypoxaemia / seizure / sepsis / organ ischaemia / metformin |
| **T**oxins | Salicylates, methanol, ethylene glycol, paraldehyde, toluene |

### Normal AG Metabolic Acidosis (NAGMA)
- **GI HCO3 loss**: diarrhoea, high drain output
- **Renal HCO3 loss**: Type 2 RTA
- **Impaired renal acid retention**: Type 1/4 RTA, renal hypoperfusion

### HAGMA Management
- Order: **BHBA + urine ketone + lactate** +/- serum/urine toxicology
- **Ketoacidosis**: rehydration (refer to H'stix / DKA section)
- **Lactic acidosis**:
  - If abdominal pain: **CT angiogram** to exclude ischaemic bowel
  - If on metformin: **stop metformin + label ADR** (MALA)

### NAGMA Management
- Check I/O; use **balanced crystalloids** (Hartmann's / Plasmalyte, NOT normal saline)
- If not explained by GI loss: workup for RTA — urine pH, urine AG

### Indications for NaHCO3
- **pH <7.15**, or **pH <7.25 with AKI**

## Metabolic Alkalosis

### Causes
- Volume / chloride depletion: vomiting, high R/T output, diuretics, Bartter syndrome
- Hyperadrenocorticoidism: Cushing's, hyperaldosteronism
- Severe hypokalaemia
- Excessive alkali intake

### Management
- Correct volume deficit with **normal saline** (most causes are chloride-responsive)
- Check RFT and **correct hypokalaemia**
- If saline-unresponsive: order **urine Cl** +/- ARR / LDDST
