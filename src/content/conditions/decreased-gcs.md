# Decreased GCS

## Causes

- **Circulatory**: hypotension/poor perfusion (HF, arrhythmia, SVCO)
- **Respiratory**: hypoxia/hypercapnia
- **Metabolic**:
  - Hypoglycaemia/hyperglycaemic crisis
  - Sepsis/hypothermia
  - Anaemia
  - Hyper/hyponatraemia, Ca, Mg, acidosis, uraemia, NH3, thiamine deficiency
  - Myxoedema coma, Addisonian crisis
  - Drug intoxication/withdrawal
- **Neurological**: head injury, CVA, seizure, CNS infection

## Urgency

- Baseline GCS + degree of drop
- **GCS <8**: airway protected? or DNACPR?
- Other vital sign abnormalities: fever, desaturation, unequal pupils, limb power drop
- Stroke history?

## Assessment

### Bedside

- Check pulse/breathing first
- Full vitals + H'stix
- **Chart GCS personally**
- Focal neurological signs:
  - Pupils, EOM/gaze, nystagmus
  - Facial asymmetry/slurring
  - 4 limbs power or posture, twitching
  - Plantar reflex
- Meningism: Brudzinski (C/I in trauma), Kernig
- CVA signs
- Status epilepticus
- ICP signs: headache, N/V, dilated pupil, CN6 palsy, papilloedema, Cushing's triad

### History

- Recent fall (especially head injury)
- Neurological symptoms
- Background: stroke post-tPA (CTB for haemorrhagic transformation), epilepsy, drug use
- Drug chart review

## Initial Management

- **ABC + DEFG** (Don't Ever Forget Glucose)
- **GCS <8**: call MO for intubation (unless DNACPR)
- Resuscitate vitals
- H'stix stat:
  - If hypoglycaemic: **IV D50 40ml stat**
  - If alcoholic/malnourished: **IV thiamine 100mg**
- NPOEM; withhold DM medications
- Neuro obs Q1H (or Q1H ×4 then Q4H if stable)

## Investigations

- **Urgent CT brain**
- CBC d/c + clotting + L/RFT + Ca/PO4 + Mg + NH3 + VBG + lactate + TnI/CK + TFT + spot cortisol
- ABG if CO2 retention
- Blood c/st if CNS infection suspected
- Serum/urine toxicology if drug intoxication suspected
- ECG, CXR
- ± LP, EEG

## Raised ICP

- Elevate HOB 30°
- Loosen neck accessories
- Urgent neurosurgical consult
- **ONLY if surgical CN3 palsy/herniation**:
  - IV mannitol (foley to BSB)
  - Hyperventilation to PaCO2 4–4.5 kPa

## Treating Underlying Causes

### Suspected CVA

- Activate stroke call
- Urgent CTB to rule out ICH
- CBC + clotting

### Status Epilepticus

- Ensure airway first
- **IV lorazepam (Ativan) 4mg** bolus, repeat once after 10–15 min
  - Or IV diazepam 5–10mg bolus
- If no IV access: IM/buccal/intranasal midazolam 10mg, or rectal diazepam 0.2mg/kg (up to 20mg)
- + Long-acting anti-seizure drug: Keppra / Dilantin / Epilim / Vimpat
- If GCS does not return to baseline within 30–60 min after jerks subside → suspect NCSE → urgent EEG

### Meningism + Fever

- Urgent blood c/st → dexamethasone + antibiotics → urgent CTB → LP
- **IV dexamethasone before/with 1st dose of antibiotics**
- IV Rocephin 2g Q12H
- IV vancomycin
- IV acyclovir 10mg/kg Q8H
- IV ampicillin 2g Q4H if immunocompromised / age >55 / alcoholic
- ± IV Flagyl if history of mastoiditis/sinusitis/otitis/abscess

### Trauma + SAH

- Urgent CT angiogram
- Aim SBP <140
- Reverse anticoagulation
- Tranexamic acid (Transamin)
- Nimodipine 60mg PO Q4H
- Urgent neurosurgical consult
