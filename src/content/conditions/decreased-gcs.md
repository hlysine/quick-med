# Decreased GCS

## Glasgow Coma Scale

| Response | Scale | Score |
| :--- | :--- | :--- |
| **Eye Opening Response** | Eyes open spontaneously | 4 Points |
| | Eyes open to verbal command, speech, or shout | 3 Points |
| | Eyes open to pain (not applied to face) | 2 Points |
| | No eye opening | 1 Point |
| **Verbal Response** | Oriented | 5 Points |
| | Confused conversation, but able to answer questions | 4 Points |
| | Inappropriate responses, words discernible | 3 Points |
| | Incomprehensible sounds or speech | 2 Points |
| | No verbal response | 1 Point |
| **Motor Response** | Obeys commands for movement | 6 Points |
| | Purposeful movement to painful stimulus | 5 Points |
| | Withdraws from pain | 4 Points |
| | Abnormal (spastic) flexion, decorticate posture | 3 Points |
| | Extensor (rigid) response, decerebrate posture | 2 Points |
| | No motor response | 1 Point |

* **Minor Brain Injury** = 13-15 points; **Moderate Brain Injury** = 9-12 points; **Severe Brain Injury** = 3-8 points

## Causes of decreased consciousness / coma
* Hypotension / poor perfusion (HF, arrhythmia, SVCO etc.)
* Hypoxia / **hypercapnia**
* **Hypoglycaemia** / hyperglycaemic crisis
* Sepsis / hypothermia
* Anaemia
* **Metabolic:** hyper/hypoNa, hyper/hypoCa, hyper/hypoMg, acidosis, uraemia, ↑ NH3, ↓ thiamine, myxoedema coma, Addisonian crisis, drug intoxication / withdrawal
* **Neurological:** HI, CVA (ischaemic/haemorrhagic), seizure (convulsive/non-convulsive), CNS infection

> **Verbal order:** vitals, neuro obs Q1H, **H'stix stat**

## How to determine the URGENCY?
* Baseline GCS and degree of drop
* If GCS < 8: **is airway protected (intubated)?** Or is it a DNACPR case if not intubated?
* Any other **vitals abnormalities** (e.g. fever, desat), **unequal pupils, limbs power drop**?
* Background history: stroke case?

## Assessment

### Bedside
* Check **pulse and breathing** for all unresponsive patients! Don't miss an arrest
* **Full set of vitals (BP/P, temp, SpO2) + H'stix**
* Chart GCS **by yourself** (*there's discrepancy from nurses' assessment more often than you think!*)
* **Focal neuro signs:** pupils, EOM/gaze, nystagmus, **facial asymmetry/slurring**, 4 limbs power (*or observe posture/spontaneous movement if cannot cooperate*), **twitching, plantar reflex**
    * **Signs of meningism (check if fever or HI):** Brudzinski (do not test if trauma), Kernig sign
    * **Signs of CVA:** gaze deviation, face/limb asymmetry, slurring, upgoing plantar
    * **Signs of status epilepticus:** focal/generalised twitching (e.g. facial/periorbital twitching / rhythmic nystagmoid eye jerks), double incontinence, up-rolling eyeballs, tongue biting
    * **S/S of ↑ ICP:** headache, N/V, dilated pupil, CN6 palsy, papilledema, Cushing's triad

### Patient records
* **Recent fall** (especially **HI / unwitnessed fall**)
* Recent neurological symptoms
* Patient background history
    * Stroke case especially post-tPA: CTB to rule out haemorrhagic transformation!
    * Epilepsy / drug use: seizure (can be **non-convulsive** status epilepticus!) / intoxication
* Drug chart

> **Tips for urgent consulting neurosurgery:**
> report **GCS (+ change in GCS) + PUPIL SIZE**

## Initial management
* **ABC and DEFG (don't ever forget glucose)**
    * If GCS < 8: call MO x **intubation** for airway protection (unless DNACPR not for intubation)
    * **Resuscitate vitals** to physiological values
    * **H'stix stat** -> if hypoglycaemic: IV D50 40ml stat
    * If alcoholic/malnourished: give **IV thiamine 100mg**
* NPOEM, W/H DM meds during fasting
* Neuro obs Q1H (or neuro obs Q1H x4 then Q4H if stable)
* **Investigations**
    * **Urgent CT brain**
    * Bld x CBC d/c, clotting, L/RFT, Ca/PO4, Mg, NH3, VBG, lactate, TnI/CK, TFT, spot cortisol
        * **+ ABG** (instead of VBG) if CO2 retention suspected
        * **+ bld c/st** if CNS infection suspected
        * + serum/urine toxicology if drug intoxication/withdrawal suspected
    * ECG LLII, CXR
    * +/- LP, EEG
* **If S/S of ↑ ICP:** elevate head of bed by 30 degrees, loosen neck accessories, **urgent consult NS**
    * ONLY if surgical CN3 palsy or herniation on CTB (or ↑ ICP but not for surgery):
        * IV mannitol (remember foley to BSB; if hypovolemia, 3% NaCl by ICU instead)
        * Hyperventilation to target PaCO2 4-4.5kPa (but reduces cerebral blood flow!)

* **Treat underlying cause\*\***
    * **Suspected CVA: activate stroke call**, + urgent CTB to r/o ICH, CBC, clotting as above
    * **Status epilepticus: ensure airway protection** before **benzodiazepines** (BZD)
        * **IV lorazepam (Ativan) 4mg bolus**, may repeat once after 10-15min
        * Or **IV diazepam (Valium) 5-10mg bolus** (*duration of action <20min, 50% recurrence within 2hrs, so must add long-acting anti-seizure drug*)
        * If no IV access: IM/buccal/intranasal midazolam 10mg, or rectal diazepam gel 0.2mg/kg up to 20mg
        * **+ long-acting anti-seizure drug** (Keppra/Dilantin/Epilim/Vimpat) **even if convulsions have ceased** (especially if diazepam/midazolam was used)
        * If GCS does not return to baseline in 30-60min after jerks subside, must suspect non-convulsive status epilepticus (NCSE) and order urgent EEG
    * **Meningism:**
        * **+ fever = meningitis: urgent bld c/st -> dexamethasone + abx -> urgent CTB -> LP**
            * **IV dexamethasone start before/with 1st dose abx** (outcome benefit for *pneumococcal, meningococcal, TB; but worse for cryptococcal!*)
            * IV Rocephin 2g Q12H (**meningitic dose**) / cefotaxime -> switch to 4th gen ceph/Meronem (Pseudomonas) if penetrating HI, neurosurgery/shunt
            * IV vancomycin (Pneumococcus, MRSA)
            * IV acyclovir 10mg/kg Q8H (VZV, HSV)
            * IV ampicillin 2g Q4H if immunocompromised/age>55/alcoholic (Listeria)
            * +/- IV Flagyl if Hx of mastoiditis/sinusitis/otitis/abscess (anaerobes)
        * **+ trauma = SAH: urgent CT angiogram**, aim SBP < 140, **reverse anticoagulation, Transamin**, nimodipine 60mg PO Q4H, **urgent consult NS**

> **Pitfall:** in suspected stroke or neurosurgical emergency, it is important to **simultaneously** activate stroke call / urgent consult NS while waiting for CTB (rather than wait for CTB first), because:
> * Even urgent CTB can take over half a day to queue!! But activating stroke protocol speeds it up
> * Earlier informing allows earlier preparation for tPA / IR thrombectomy / OT – time is brain!

> **Tips for commenting CT brain:** the most common template is **"no ICH/SOL/MLS"**
> **Compare both sides & correlate with clinical picture!** Important things to look for:
> * **↑ ICP:** midline shift, compressed ventricles/sulci, hydrocephalus, uncal/tonsillar herniation
> * **ICH, EDH, SDH** (*pay attention to falx*), **SAH** (*pay attention to suprasellar/ambient cisterns*)
> * If suspected HI: hematoma,air fluid level in paranasal sinuses, **fracture on bone window**
> * **Ischaemic stroke:** hyperdense MCA / basilar dot sign, loss of grey-white differentiation, infarct
