# Hypotension

## Causes

- Hypovolemia
  - Hemorrhage
  - Renal/extra-renal/third-spacing
- Distributive
  - Sepsis
  - Anaphylaxis
  - Addisionian crisis
  - Neurogenic shock
- Cardiogenic
  - Arrhythmia
  - ACS
  - Congestive heart failure
  - Pericarditis/effusion
- Obstructive
  - Pulmonary embolism
  - Tamponade
  - Tension pneumothorax

## Assessment

- Determine urgency
  - How low is BP compared with baseline?,
  - Tachycardia? Alertness? Symptoms?
  - DNACPR?
    - May be not for inotropes/fluid challenge
  - On antihypertensives?
- Bedside
  - Recheck vitals, HR, RR
  - GCS/alertness
  - Chest pain/SOB
    - Cardiogenic/obstructive
  - Elevated JVP
    - Cardiogenic/obstructive
  - Warm peripheries
    - Septic
  - Tender calves
    - Obstructive
  - Fluid status, basal crep, LL edema, skin turgor, mucous membrane
  - Abdomen exam, PR, drain output
    - Hypovolemic
- History
  - Cardiac/renal function (Cr, LVEF)
  - Antihypertensives
  - Steroids (Addisonian crisis)
  - Current medical problems

## Management

### Monitoring

- BP/P/SpO2
  - If mild: Obs Q1hx2 then Q4h if stable
- Foley to BSB, chart UO Q1h/Q4h

### Investigations

- ECG, CXR
- Bloods: CBC, LRFT, Clotting
  - If ACS: TnI/CK
  - If sepsis: blood/urine/sputum c/st
  - If bleeding: endoscopy/imaging (per MO)
  - If Addisonian crisis: spot cortisol, paired ACTH, random glucose, VBG, RFT

### Fluid resuscitation

- Gelofusine/Hartmann/NS 500ml FR/Q1h x1
- 250ml if fluid overloaded

### Inotropes

- Call MO before starting

### Meds

- Withhold antihypertensives & diuretics if SBP <110
