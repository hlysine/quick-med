# Hypotension

## Causes

- Hypovolemia
  - Hemorrhage
  - Renal/extra-renal/third-spacing
- Distributive
  - Sepsis
  - Anaphylaxis
  - Addisionian crisis
  - Neurogenic shock
- Cardiogenic
  - Arrhythmia
  - ACS
  - Congestive heart failure
  - Pericarditis/effusion
- Obstructive
  - Pulmonary embolism
  - Tamponade
  - Tension pneumothorax

| Type             | Causes                                                       | Heart rate                                                                                            | JVP         | Peripheries |
|------------------|--------------------------------------------------------------|-------------------------------------------------------------------------------------------------------|-------------|-------------|
| **Cardiogenic**  | Myocardial infarction<br>Arrhythmia                          | **Heart failure**<br>Moderate&nbsp;tachycardia<br>**Arrhythmia**<br>Severe bradycardia or tachycardia | High/normal | Cold        |
| **Hypovolemic**  | Hemorrhage<br>Dehydration                                    | Moderate tachycardia                                                                                  | Low         | Cold        |
| **Distributive** | Sepsis<br>Anaphylaxis<br>Addisonian crisis                   | Moderate tachycardia                                                                                  | Low         | Warm        |
| **Obstructive**  | Massive PE<br>Cardiac tamponade<br>Tension&nbsp;pneumothorax | Moderate tachycardia                                                                                  | Very high   | Cold        |

## Assessment

- Determine urgency
  - How low is BP compared with baseline?,
  - Tachycardia? Alertness? Symptoms?
  - DNACPR?
    - May be not for inotropes/fluid challenge
  - On antihypertensives?
- Bedside
  - Recheck vitals, HR, RR
  - GCS/alertness
  - Chest pain/SOB
    - Cardiogenic/obstructive
  - Elevated JVP
    - Cardiogenic/obstructive
  - Warm peripheries
    - Septic
  - Tender calves
    - Obstructive
  - Fluid status, basal crep, LL edema, skin turgor, mucous membrane
  - Abdomen exam, PR, drain output
    - Hypovolemic
- History
  - Cardiac/renal function (Cr, LVEF)
  - Antihypertensives
  - Steroids (Addisonian crisis)
  - Current medical problems

## Management

### Verbal orders

- ECG, CXR
- IVF
  - 250ml FR/Q1h if fluid overloaded
  - 500ml FR if not fluid overloaded

### Monitoring

- BP/P/SpO2
  - If mild: Obs Q1hx2 then Q4h if stable
- Foley to BSB, chart UO Q1h/Q4h

### Investigations

- ECG, CXR
- Bloods: CBC, LRFT, Clotting
  - If ACS: TnI/CK
  - If sepsis: blood/urine/sputum c/st
  - If bleeding: endoscopy/imaging (per MO)
  - If Addisonian crisis: spot cortisol, paired ACTH, random glucose, VBG, RFT

### Fluid resuscitation

- Gelofusine/Hartmann/NS 500ml FR/Q1h x1
- 250ml if fluid overloaded

### Inotropes

- Call MO before starting

### Meds

- Withhold antihypertensives & diuretics if SBP <110


### Treat underlying cause

**Sepsis**

- Antibiotics - IV Augmentin / Tazocin / Meropenem
- Source control

**Addisonian crisis**

- H'stix Q4h
- NS Q1h x4-6
- IV hydrocortisone 100mg Q8h / 50mg Q6h

**Anaphylaxis**

- IM adrenaline 1:1000 0.5mg stat
- Remove allergen
- IV hydrocortisone 100mg Q6-8h
- IV Piriton 10mg Q8h

**Hypovolemia**

- IVF
- Rule out active bleeding

**Bleeding**

- Type & screen, transfusion
- Withhold antiplatelets
  - Consult cardiac for DAPT
- Withhold/reverse anticoagulation
- Stop bleeding

**Congestive heart failure**

- Withhold lasix/TNG if SBP <100
- Withhold beta blocker/ACEI
- Treat arrhythmia/ischemia/structural lesions

**Massive PE/tamponade**

- Consult CCU
- Urgent echo
- Thrombolysis/pericardiocentesis

**Tension pneumothorax**

- Immediate needle decompression
  - 16-18G needle at 2nd intercostal space midclavicular line
- Chest drain afterwards
