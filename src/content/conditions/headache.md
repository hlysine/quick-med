# Headache

## Causes
* Primary: tension headache, migraine, cluster headache
* Secondary: **CNS infection, SOL, ICH/SAH/SDH/EDH**, cerebral venous sinus thrombosis, carotid/vertebral artery **dissection, giant cell arteritis, glaucoma**, trigeminal neuralgia, TMJ/odontogenic/cervicogenic headache

## Assessment
* Vitals (*if fever + headache, manage as CNS infection, need bld c/st & LP*)
* **GCS**
* Recent **fall/HI**
* SOCRATES of headache – **Thunderclap** (SAH/dissection)? **Worse when erect/supine** (↑ ICP/↓ ICP)?
* Associated symptoms: **fever, neck/eye pain, visual disturbance**, jaw claudication, neuro symptoms
* **Full neuro exam:** CN (pupils, EOM, CN7, dysarthria), 4 limbs power, cerebellar signs, **meningism**
* Background history: cancer, immunosuppression, **coagulopathy, antiplatelet/anticoagulant**

## Management
* Panadol prn
* If **red flags** present: Neuro obs, urgent CT brain, +/- LP
* If **S/S of ↑ ICP:** elevate head of bed by 30 degrees, loosen neck accessories, **urgent consult NS**
    * ONLY if surgical CN3 palsy or herniation on CTB (or ↑ ICP but not for surgery):
        * IV mannitol (remember foley to BSB; if hypovolemia, 3% NaCl by ICU instead)
        * Hyperventilation to target PaCO2 4-4.5kPa (but reduces cerebral blood flow!)
* If S/S of ↓ ICP: lie flat, IVF replacement
* If **fever**: **urgent bld c/st -> dexamethasone + abx -> urgent CTB + LP**
    * IV dexamethasone start before/with 1st dose abx (*outcome benefit for pneumococcal, meningococcal, TB; but worse for cryptococcal!*)
    * IV Rocephin 2g Q12H (**meningitic dose**) / cefotaxime -> switch to 4th gen ceph/Meronem (Pseudomonas) if penetrating HI, neurosurgery/shunt
    * IV vancomycin (Pneumococcus, MRSA)
    * IV acyclovir 10mg/kg Q8H (VZV, HSV)
    * IV ampicillin 2g Q4H if immunocompromised/age>55/alcoholic (Listeria)
    * +/- IV Flagyl if Hx of mastaoditis/sinusitis/otitis/abscess (anaerobes)
* If suspect **SAH: urgent CT angiogram, aim SBP < 140, reverse anticoagulation, Transamin**, nimodipine 60mg PO Q4H, **urgent consult NS**
* If **new or enlarged haematoma/ICH:** aim SBP < 140, Plt > 100, INR < 1.5, +/- urgent consult NS
* If **artery dissection: activate stroke call (if stroke), urgent consult NS**, + urgent CT angiogram
* If **GCA:** systemic glucocorticoid, urgent consult OVS
* Consult neuro if persistent
