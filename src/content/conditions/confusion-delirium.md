## Causes

| Category | Examples |
|---|---|
| **Haemodynamic/perfusion** | Dehydration/hypotension, HF, arrhythmia, SVCO, hypertensive encephalopathy |
| **Respiratory** | Hypoxia, hypercapnia |
| **Metabolic — glucose** | Hypoglycaemia, hyperglycaemic crisis |
| **GI/pain** | AROU, constipation, pain |
| **Major stress** | Sepsis, ACS/PE/dissection, acute abdomen, haemorrhage |
| **Metabolic — electrolyte/organ** | Hyper/hyponatraemia, hypercalcaemia, hypo/hypermagnesaemia, acidosis, uraemia, hyperammonaemia, thiamine deficiency, folate/B12 deficiency, myxoedema coma, Addisonian crisis, drug intoxication/withdrawal |
| **Neurological** | HI, CVA, seizure, CNS infection |
| **Psychiatric** | Dementia, psychosis |

## Assessment

- Vitals — if **fever + confusion**: manage as CNS infection → blood c/st + LP
- **GCS + H'stix stat**
- I/O chart
- Full neurological exam: CN, 4-limb power, cerebellar signs, meningism
- General exam: septic source, respiratory distress, abdominal tenderness, calf tenderness
- Drug/alcohol history

## Management

Select relevant items based on clinical suspicion.

### Environmental and General

- Environmental cues: hearing/visual aids, side room near window, clock, familiar visitors
- Chart I/O, maintain hydration
- **PO/IV thiamine**

### Behavioural

- **PO/IV Haldol (haloperidol) 0.5 mg** as directed prn (max 5 mg/day) for agitation/psychosis
- Cease offending drugs: benzodiazepines (except in alcohol withdrawal), anticholinergics, opioids

### Investigations

- **CT brain** (must order if no obvious cause)
- **Bloods**: CBC d/c + clotting + L/RFT + Ca/PO4 + Mg + NH3 + VBG + lactate + TnI/CK + TFT + B12 + folate + spot cortisol; ABG if CO2 retention; blood c/st if sepsis/CNS infection suspected; serum toxicology if drug intoxication/withdrawal
- **MSU/CSU**: m'stix, c/st (+ toxicology if drug intoxication)
- **Sputum** c/st if any
- **PVRU** × 1
- CXR, KUB, ECG
- ± LP, EEG

### Referrals

- Consult OT for cognitive assessment
- Consult MED / Geriatrics / Psychiatry as appropriate
