# H'stix Abnormalities

> **Usually nurses will inform you when H'stix < 4 or > 12**

## How to determine the URGENCY?
* **Hypoglycaemia:** should be treated with glucose (oral or IV) **immediately** – prescribe remotely then assess later!
* **Hyperglycaemia:** very common; not too urgent unless unconscious or symptoms of DKA/HHS

## Hyperglycaemia

### Information you would like to know
* Is the patient **NPO**? If not, is it pre-meal or post-meal H'stix?
* Any **regular DM meds**? Are they resumed?
* Any **S/S of DKA/HHS**? E.g. abd pain, N/V, Kussmaul breathing, dehydration, confusion, ↓ GC

### Initial management
* **H'stix TDS + nocte** (or H'stix Q4H if >20 or symptomatic)
* **SC Actrapid stat** (+/- sliding scale matching the frequency of H'stix monitoring)
    * Each dept's sliding scale template varies (e.g. 4 units if H'stix 16-19.9, 6 units if H'stix 20-23.9, 8 units if H'stix 24-27.9, 10 units if H'stix >=28), some are more aggressive and some looser
    * **Treat less aggressively if: 1) before bedtime, 2) NPO, 3) renal failure (impaired insulin excretion), 4) on regular mixed insulin (contains short acting) -> to prevent hypoglycaemia**
* **If diet allowed:** DM diet, resume usual DM meds (**check Cr, note renal dosing!!**)
    * Do not resume SGLT2 inhibitors during stress episode (usual practice: resume upon discharge)

### Persistent hyperglycaemia
* **Bld x FG, HbA1c**
* **Refer GOPC** +/- consult MED x poor DM control / newly diagnosed DM
* **Consider start metformin 500mg daily** if no C/I
    * Max daily dose 2000mg
    * Max daily dose 1000mg if eGFR 30-44
    * **C/I if eGFR <30 / liver failure / metabolic acidosis / tissue hypoxia (HF, resp failure, PE, sepsis) / alcoholism**

## Hyperglycaemic emergencies – DKA/HHS

| Feature | DKA | HHS / HONK |
| :--- | :--- | :--- |
| **S/S** | Abd pain, vomiting, dehydration, Kussmaul breathing, fruity odour | Confusion, ↓ GC, dehydration |
| **Diagnosis / Investigations** | **D:** H'stix/RG >14<br>**K:** ketonemia >3 / ketonuria >2+<br>**A:** pH <7.3, HCO3 <15<br><br>+ workup for ddx of abd pain/vomiting | **H:** H'stix/RG >30-33<br>**H:** serum osmo >320<br>**Non-ketotic:** pH >7.3, HCO3 >15, no/mild ketoneuria/ketonemia<br><br>or altered mental state |
| **Management** | H'stix **Q1H**, aim H'stix 6-11<br>Bld x RG, VBG, anion gap, beta-hydroxybutyric acid (BHBA/BOHB), RFT **Q4-6H**<br>Urine x m'stix, ketone<br>**Rehydration:** IVF Q1H x2 then maintenance IVF Q6H, chart I/O<br>• Use NS; switch to 1/2:1/2 or D5 when hyperNa or H'stix < 14<br>• + KCl into IVF as needed to keep K normal<br>**Insulin pump:** Actrapid 50 units in 50ml NS/gelofusine, start at 0.1 unit/kg/hr, titrate as variable or fixed rate, aim H'stix 6-11 (refer to your hospital's template)<br>**Urgent consult endocrine** +/- ICU |

> **Pitfall:** in **euglycemic** DKA, H'stix is not high! So for patients on **SGLT2 inhibitor, acidosis / urine ketone +ve** should prompt you to workup for DKA!

## Hypoglycaemia
* **Immediate glucose administration**
    * If patient can eat and borderline H'stix 3-4: apple juice / sugar granules
    * If NPO or severe hypogly: **IV Dextrose 50% (D50) 40ml bolus**
    * Consider giving IV thiamine 100mg bolus in alcoholics/poor intake
* **If severe hypoglycaemia (e.g. H'stix <2 or ↓ GCS)**
    * **W/H all DM meds including insulin**
    * **Maintenance IV D10 infusion Q6-8H** (*especially if originally on long-acting DM meds!*)
    * **Monitoring:** recheck H'stix in 15 min + H'stix Q1H x4 then Q4H if stable
* **Down-titrate DM meds** (check **Cr level** and note renal dosing!)
* **Recurrent hypoglycaemia: consult endocrine** x DM drug titration / workup for non-DM patients (e.g. liver failure, hyperinsulinism, glucagon/cortisol/epinephrine deficiency)
