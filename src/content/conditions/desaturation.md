# Desaturation

## Definition

- SpO2 <92%
- Most urgent cause after cardiac arrest

## Causes

- **Upper airway**: obstruction, sputum retention, aspiration
- **Lungs**: PTX, pleural effusion/collapse, pneumonia, asthma/COPD/ILD/bronchiectasis exacerbation
- **Cardiovascular**: PE, ACS, tamponade, APO
- **Neurological**: hypoventilation/apnoea

## Urgency

- Degree of desat + O2 requirement
  - 0 → 2L: minor
  - 0 → 6L or 6 → 10L: major
- **Attend immediately** if SpO2 still <88% despite up-titrating O2
- Assess: BP and HR, alertness

## First Step: Titrate O2

- SpO2 target >92% (88–92% in COPD)

| Delivery Device | Flow Rate | FiO2 |
|---|---|---|
| Nasal cannula | 0–6 L/min | 0.24–0.4 |
| Simple Hudson mask | 5–10 L/min | 0.35–0.5 |
| Non-rebreathing mask | 10–15 L/min | 0.7–1.0 |
| Venturi mask | Variable | Near-fixed FiO2 |
| BVM (Ambubag) | — | 1.0 |

- **BVM technique**: tight seal E-C clamp; RR 12–16, TV 300–500ml (¼ bag); if high resistance suspect airway obstruction; avoid intubating alone
- **If ≥10L**: call MO to discuss intubation/PAP/DNACPR

### Invasive Ventilation

- Mode: SIMV; RR 12–16; TV 6–8 ml/kg IBW; FiO2 titrate from 1.0; PEEP 5
- I:E ratio: 1:2 (1:3 for obstructive; 1:1.5 for poor compliance/ARDS)
- Sedation: Dormicum 45mg in 45ml NS, start 1 ml/hr; withhold if SBP <90
- Repeat ABG 1 hr after

### BiPAP

- For T2RF with respiratory acidosis (check NIV C/I first)
- Settings: IPAP 14, EPAP 4, RR 14, FiO2 0.4
- Repeat ABG 1 hr after

### CPAP

- For APO or OSA

### HFNC

- For T1RF; start 60 L/min, FiO2 0.3–0.6, flow 30–60, temp 34°C

## NIV Contraindications (Need Intubation)

- **A** – Fixed airway obstruction / excessive secretions
- **B** – Frank/impending respiratory arrest, undrained PTX
- **C** – Hypotension, arrhythmia, cardiac arrest
- **D** – GCS <8
- **E** – Facial trauma/burns

## Assessment

### Bedside

- Full vitals + GCS
- Signs of airway obstruction, choking, cough/sputum, haemoptysis
- Respiratory exam: asymmetry, trachea deviation, breath sounds
- Calves (DVT), fluid status

### History

- Cardiac, pulmonary, neurological disease

## Initial Management

- NPOEM if significant desat or aspiration
- Prop up (sit upright)
- O2 as needed
- Monitoring: BP/P/SpO2 Q1H, continuous SpO2

## Investigations

- CXR, ECG
- CBC d/c + L/RFT + TnI/CK + clotting
- D-dimer if CTPA planned
- ABG if significant desat or T2RF
- Microbiology: sputum c/st, NPS (flu/COVID), urine pneumococcal/legionella antigen
- Spirometry for neurological causes (early ventilation if FVC <15 ml/kg)
- CT thorax with contrast for unknown cause

## Treating Underlying Causes

### Anaphylaxis

- IM adrenaline 1:1000 0.5ml ± repeated
- Remove allergen
- IV hydrocortisone 100mg Q6–8H
- IV Piriton 10mg Q8H
- **AVOID sedation** — need awake fibreoptic intubation / tracheostomy / cricothyroidotomy

### Aspiration

- Speech therapist referral
- Sputum c/st
- Antibiotics
- If +dysarthria/dysphagia: consider bulbar/pseudobulbar palsy

### Sputum Plugging / Atelectasis

- Chest physiotherapy

### Pneumothorax

- Chest drain (BTS algorithm)
- Plt >50 + INR <1.5 before insertion
- C/I for ventilation before drain is placed

### Massive Pleural Effusion

- Pleural tap ± pigtail drain
- Send fluid: cell count, TP, LDH, pH, glucose, ADA, AFB, TB culture, c/st, cytology (with paired blood)

### APO

- Off IVF (even if NPO); FR <1.2 L/day + low salt
- IV Lasix 40mg stat + 20mg Q12H (withhold if SBP <100)
- On PD: extra 4.25% PDF Q2H ± IP Actrapid 4–8 U/bag if DM

### Chest Infection

- Antibiotics:
  - Mild/moderate CAP: IV/PO Augmentin + PO doxycycline
  - HAP / severe CAP / bronchiectasis: IV Tazocin
  - Quinolone if beta-lactam allergy
  - Immunocompromised: consider TB, PCP, HIV, Candida

### Wheeze

- Ventolin 4 puffs Q4H prn
- Atrovent 2 puffs Q4H prn (max 12 puffs/day)

### Asthma / COPD Exacerbation

- Ventolin 4 puffs up to Q1H + Atrovent
- IV hydrocortisone 100mg Q6H or PO prednisolone 40mg daily
- **Asthma**: + IV MgSO4 1.2–2g over 20 min
- **COPD**: + IV/PO Augmentin if ≥2 symptoms
- ABG; ventilatory support if respiratory acidosis

### PE

- SC enoxaparin 1 mg/kg Q12H (Q24H if CrCl <30)
- If obstructive shock: refer to hypotension section

## Desaturation on Benzodiazepine

- Stop BZD
- Rule out anaphylaxis
- Airway maneuvers + BVM
- ± IV flumazenil 0.2mg Q1min (up to 4 times)
- Obs Q20min ×3

## Desaturation on Opioid

- Stop opioid
- Rule out anaphylaxis
- Airway maneuvers + BVM
- IV/IM/SC/ET naloxone 0.1–0.2mg Q2–3min
- Obs Q1H
