# Fever

## Definition

- Temp >38°C

## Urgency

- Not usually urgent unless vitals unstable (tachycardia, shock, desaturation) or **neutropenic**

## Assessment

### Bedside

- Full vitals
- Localising symptoms: URTI, LRTI, UTI, GI, hepatobiliary/pancreatobiliary, OG, CNS, skin, joints
- Do not miss: catheter sites/drain output, rash, meningism, calves (DVT)

### History

- Background, previous septic workup, past c/st results, current antibiotics

## Initial Management

### Septic Workup

- CBC d/c + L/RFT + CRP ± TFT ± VBG + lactate
- **Blood c/st** if >38°C (no repeat if taken within 3 days or result pending)
  - On central line: paired c/st (1 peripheral + 1 from each CVC lumen)
- CXR
- Sputum c/st ± AFB
- ± NPS for respiratory viruses
- Urine m'stix + c/st
- ± Stool (c/st, viral PCR, ova/cysts, C. diff if admission >96hrs / recent antibiotics / IBD)
- ± Wound/skin swab
- ± Imaging
- ± LP/biopsy: cell count, gram stain, c/st

### Symptomatic Relief

- Panadol prn (avoid if elevated ALT)
- Ice pack prn

### Antibiotics

- Start/step-up based on clinical suspicion
- Check past c/st, allergy, CrCl
- Common empirical escalation: IV Augmentin → Tazocin → Meronem
  - Penicillin allergy: levofloxacin

### Source Control

- Foreign body removal, drainage, debridement

### Other

- Withhold immunosuppressants
- Give stress dose (double usual) if on long-term steroids

## Empirical Antibiotics — Special Cases

| Indication | Regimen |
|---|---|
| CAP / STI | + Doxycycline or azithromycin |
| PID / liver abscess / gas-forming infection | + Flagyl |
| CNS infection | IV Rocephin 2g Q12H + vancomycin + acyclovir ± ampicillin ± Flagyl |
| MSK infection | Cefazolin / ampicillin + cloxacillin |
| SBP | Cefotaxime / Rocephin / norfloxacin / ciprofloxacin |
| CAPD peritonitis | IP cefazolin + ceftazidime; IV cefazolin + cefepime/gentamicin if septic |

## Neutropenic Fever

- **Definition**: ANC <1.0 ×10⁹/L + temp ≥38.3°C once, or >38°C sustained for 1 hr
- Reverse isolation; neutropenic diet; close monitoring
- **Full septic workup**: URGENT blood c/st (paired if CVC)
- After cultures, **promptly start antibiotics within 1 hr** with anti-pseudomonal coverage: IV Tazocin + stat
- Consider opportunistic infections: TB, VZV/CMV, fungi
- Prompt source control
- ± G-CSF (off-label; **C/I in myeloid neoplasm**): Nivestim 30 million units Q24H SC
