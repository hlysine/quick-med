# Fever

> **Usually nurses will inform you when Temp > 38°C**

## How urgent is it?
Fever alone is usually not urgent unless: 1) vitals unstable (e.g. tachy, shock, desat), 2) **neutropenic**

Most units expect you to perform full septic workup when temp > 38°C, and teaching hospitals in particular may want you to take bld c/st before starting antibiotics.

## Assessment

### Bedside
* Vitals
* **Localising s/s:** URTI, LRTI, UTI, GI, HBP, OG, CNS, skin, joints
    * **Do not miss: catheter sites/drain output, rash, meningism, calves (DVT)**

### Patient records
* Patient background history
* Whether previous septic work-up has already been done
    * Check **past c/st results** (including recent admissions which may guide antibiotics use)
* Recent and current antibiotics

## Initial management
* **Septic work-up**
    * Bld x CBC d/c, L/RFT, CRP, +/- TFT, +/- VBG, lactate
    * Bld c/st if >38°C (no need to repeat if taken within 3 days or result still T/F)
        * **If on central line: paired c/st** (1 set from peripheral, 1 set from each CVC lumen)
    * CXR
    * Sputum x c/st +/- AFB if any
    * +/- NPS x resp viruses (e.g. flu, COVID) if URTI symptoms
    * Urine x m'stix, c/st
    * +/- Stool x c/st, viral PCR, ova/cysts, C diff if diarrhea
    * +/- Wound/skin swab / pus / fluid etc. x c/st
    * +/- Imaging based on localising s/s e.g. CT brain/A+P, USG urinary/hepatobiliary system
    * +/- Diagnostic tapping/LP/biopsy x cell count, gram stain, c/st
* Panadol prn (avoid if ALT ↑), ice pack prn
* **Start / step-up antibiotics based on clinical suspicion**
    * **Check relevant past c/st results, drug allergy, CrCl (renal dosing)**
    * Common empirical abx: IV Augmentin -> Tazocin -> Meronem (Penicillin allergy: levofloxacin)
* **Source control** if known e.g. removal of FB, drainage, debridement
* **W/H immunosuppressants – but give stress dose (double usual dose) if on long-term steroid!**

> **Empirical abx choice depends on clinical suspicion (*not always Augmentin!*) Special cases:**
> * CAP / STI: + doxycycline/azithromycin
> * PID / liver abscess / gas forming: + Flagyl
> * CNS: IV Rocephin **2g** Q12H (meningitic dose) + vancomycin + acyclovir +/- ampicillin +/- Flagyl
> * MSK: cefazolin / ampicillin+cloxacillin
> * SBP: cefotaxime/Rocephin / norfloxacin/ciprofloxacin
> * CAPD peritonitis: IP cefazolin + ceftazidime; IV cefazolin + cefepime/gentamicin if septic
>
> *Refer to **IMPACT HK** guidelines! And **basing abx choice on relevant past c/st results is the best!***

## Common Pitfalls – NEUTROPENIC FEVER
* Definition: **absolute neutrophil count (ANC) < 1.0 x 10^9/L** + 38.3C once or >38C for 1hr
* Vulnerable patients: on chemotherapy / haematological malignancy
    * These patients often have a central venous catheter (CVC) – watch out for **line sepsis!**
* Presentation can be subtle due to immunosuppression

### Management
* Reverse isolation (usually can only place in side room)
* Neutropenic diet
* Close monitoring of vitals if unstable
* Full septic workup as above: **URGENT bld c/st** (paired c/st if on CVC)
* *After* bld c/st, promptly start abx (within 1hr) with **anti-pseudomonal coverage – IV Tazocin + stat**
    * Consider opportunistic infections (TB, VZV/CMV, fungi)
* Prompt **source control**
* +/- G-CSF (off label use; C/I in myeloid neoplasm; leave to MO!): Nivestim 30 million-units Q24H SC

> Haematology patients may have a "fever plan" made in advance – so just take blood c/st and follow the abx plan!

### Tips on taking paired bld c/st (1 set from each CVC lumen + 1 set from peripheral vein)
* **Equal volume for all sets:** inoculation volume affects the *time to positive c/st*, which is used to determine likelihood of line infection (**hence whether the line needs to be removed!**)
* **Clearly label which lumen** the sample was from: abx will be infused into that specific lumen
* **Technique of blood taking from CVC:** good line care **prevents line infection/blockage!**
    * Stop IVF for 3min, TPN for 5min before blood taking (prevent drip arm effect)
    * Aseptic technique: on sterile set, prepare **per lumen** – 1 chlorhexidine wipe, 4 10ml syringes (for aerobic sample, anaerobic sample, NS flush, heparin flush), 10ml NS, 5ml heparin (conc. heparin for HD cath; heparinised saline for hemat patients)
    * Rub connecting hub **very hard for 15s** with chlorhexidine wipe
    * Discard first 5ml of blood if not taking bld c/st; but no need to discard for bld c/st
    * After taking blood: flush with 10ml NS using **push-pause-push-pause** technique (create turbulent flow to dislodge clots/platelets/fibrin); then flush with 5ml heparin (conc. heparin for HD cath; heparinised saline for hemat patients), create **positive pressure** (push hard and fast and close the clamp while flushing, to prevent back flow of blood which may clot lumen)
    * Remember to **close clamp** (don't leave catheter blood open to air)
    * Apply disinfecting cap if a/v (can cap with combi stopper though it doesn't prevent line infection)
