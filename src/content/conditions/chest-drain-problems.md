- **Indications for chest drains**: pneumothorax, pleural effusion, empyema

- **Normal functioning chest drain**: swinging, underwater seal, bubbling (if PTX), may/may not need suction

- **No bubbling, no swinging**:
  - Causes: fully expanded PTX/fully drained effusion (drain holes appose pleura), blocked/kinked drain, dislodgement
  - Management: check patient status (dyspnoea, SpO2, O2 requirement); visible blockage/kinking/dislodgement
    - If blocked/kinked: check for subcutaneous emphysema
    - If subcutaneous emphysema: remove blockage/kinking, increase suction/reinsert larger drain; if stridor/desat = need skin incision
  - CXR to confirm; if clinically well + lung re-expanded + drain in-situ: remove chest drain (keep if pleurodesis needed)

- **Air leak (bubbling)**:
  - System air leak = no bubbling/swing after suction off
  - Patient air leak = increases during cough/speech
  - Dislodgement (check last side hole vs chest cavity on CXR); try pinching skin at exit, add suture; check connections
  - If persistent air leak >3-5 days for PTX: CT thorax + consult CTS

- **Haemothorax**:
  - Inspect for clots (fresh bleeding = clots); complications: desat (mass effect), shock, anaemia
  - Management: obs Q1H; large bore (32-36Fr) chest drain; haemodynamic support + transfusion; blood x CBC + clotting; Transamin if no C/I
  - **Urgent consult CTS** if massive haemothorax (initial blood loss >1.5L or continuing >200ml/hr ×4hrs)
  - **DO NOT spigot chest drain even if large bleed**

- **Indications for pleurodesis**: traumatic PTX, recurrent PTX (1st contralateral or 2nd ipsilateral), bilateral PTX, secondary spontaneous PTX, spontaneous haemothorax, persistent air leak, failed lung re-expansion, high-risk occupations, pregnancy
