# Palpitations

## Causes
* Caffeine, alcohol, drug/withdrawal
* Arrhythmia
* **Hypoglycaemia**
* **Anaemia**
* **Sepsis**
* Thyrotoxicosis
* Phaeochromocytoma
* Pregnancy, perimenopause
* Anxiety/panic

## Assessment
* **Vitals, pulse regularity**
* Onset, character (fast vs slow, regular vs irregular)
* **Cardiac/anaemic symptoms: chest discomfort, SOB, dizziness, LOC**
* **Autonomic symptoms:** tremor, sweating, heat intolerance, diarrhoea, amenorrhoea
    * *DDx hypoglycaemic/ thyrotoxic/ phaeochromocytoma/ climacteric*
* Anxiety/panic
* Caffeine, alcohol, drug intake

## Management
* H'stix monitoring
* Cardiac monitor (especially if true tachycardia)
* ECG LLII
* CBC, RFT, Ca/PO4, TFT, (24hr urine metanephrines only if headache + autonomic symptoms)
* Treat underlying cause; may need to consult/refer MED for arrhythmia/endocrinopathy
