title: Acute Retention of Urine (AROU)

- **Definition**: difficult painful voiding with PVRU >300-350ml

- **Causes**:
  - Bladder outflow obstruction: constipation, blood clot, bladder stone, urethral stricture
    - Male: BPH/CA prostate/bladder neck/phimosis/paraphimosis
    - Female: pelvic mass, pelvic organ prolapse
  - Neurogenic bladder: cord/conus/cauda compression, autonomic neuropathy (DM/GBS)
  - Drugs: GA/EA, anticholinergic, opioid, decongestant

- **Complications**: UTI, obstructive uropathy (post-renal AKI), post-obstructive diuresis

- **Assessment**:
  - Urinary symptoms (haematuria, dysuria, loin pain, fever)
  - Reversible causes (post-op, pain, constipation)
  - LL power (r/o neurological compression), neuro symptoms
  - PR exam (prostate size/contour/median sulcus/nodules; anal tone/grip/perianal sensation; faecal loaded)
  - Background (baseline LUTS, previous DRE/PSA/prostate sizing/biopsy, neuro disease)

- **Management**:
  - Foley to BSB, chart U/O Q4H (watch for post-obstructive diuresis)
  - Some depts prefer cath once then PVRU ×1, foley to BSB if >350ml
  - If likely reversible: TWOC 24-48hrs later + PVRU ×1 + reinsert if >350ml
  - Monitor CBC d/c + RFT; CSU x m'stix + c/st; ±KUB (bladder stone)
  - Off offending drugs; laxatives + BO daily; analgesics + good pain control
  - If BPH not on alpha blocker: **Xatral XL 10mg nocte** if BP tolerates (C/I in postural hypotension, severe renal/hepatic impairment)
  - If refractory on alpha blocker: indicated for surgery
  - Consult URO if failed TWOC or recurrent AROU
