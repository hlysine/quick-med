# Dysuria

## Causes
* Cystitis (UTI)
* Urethritis (STI)
* Stone
* Foley irritation (not true dysuria)

## Assessment
* **Fever**
* **Urinary symptoms:** renal colic (radiate to suprapubic area), LUTS, dysuria, haematuria, turbid urine, passage of stone
* **STI symptoms:** urethral discharge, penile/vulval mass/ulcer
* **Sexual history**
* **Abdominal exam, costovertebral tenderness**

## Management
* **MSU/CSU x m'stix, c/st**
* If on foley: **change foley**, inflate new foley balloon to 10ml only (minimise trigone irritation)
* Empirical antibiotics (**according to past urine c/st**) – Augmentin/ciprofloxacin/Septrin/gentamicin
    * Ertapenem or meropenem if history of ESBL
    * Avoid nitrofurantoin as bacteriostatic only (vs others are bactericidal)
* **If recent sexual exposure or s/s of STI:**
    * Bld x HBsAg, anti-HCV, anti-HIV, VDRL; first-catch urine / high vaginal swab x gonorrhoea, chlamydia, trichomonas
    * + PO doxycycline x 1 week or PO azithromycin x 1 dose
    * Consult URO/GYN
* **PO pyridium** for symptomatic relief (but not useful for foley irritation)

> **Interpretation and management of urine m'stix results:**
> * **WBC +ve (pyuria)** indicates an inflammatory process, non-specific, can be infective or non-infective
> * **Nitrite +ve** (presence of nitrite-reducing bacteria) indicates bacteriuria (UTI vs colonisation); only need to treat if 1) symptomatic 2) pregnant 3) transplant kidney 4) recent urological procedures, but in practice we usually prescribe antibiotics if nitrite +ve or c/st +ve
> * **RBC +ve (haematuria):** workup for UTI, stones, CA, GN, or rhabdomyolysis (myoglobinuria)
> * **Sugar +ve (glycosuria):** if no known DM, order FG + HbA1c, monitor H'stix
> * **Ketone +ve (ketonuria):** ddx starvation, AKA, DKA; 1) ↑ IVF, 2) monitor H'stix, 3) if H'stix ↑ or on SGLT2i (even if euglycemic), workup for DKA (RG, VBG, BHBA, urine ketone)
> * **Protein +ve (proteinuria):** ddx UTI, CKD, nephrotic syndrome; order urine c/st and UPCR; if DM, order UACR, may need ACEI if microalbuminuria
