- **Causes**: cystitis (UTI), urethritis (STI), stone, foley irritation

- **Assessment**: fever; urinary symptoms (renal colic, LUTS, haematuria, turbid urine, passage of stone); STI symptoms (urethral discharge, penile/vulval mass/ulcer); sexual history; abdominal exam, costovertebral tenderness

- **Management**:
  - MSU/CSU x m'stix + c/st
  - If on foley: change foley, inflate new balloon to 10ml only (minimise trigone irritation)
  - Empirical antibiotics (per past urine c/st): Augmentin/ciprofloxacin/Septrin/gentamicin; **Ertapenem/meropenem if ESBL history**; avoid nitrofurantoin (bacteriostatic only)
  - If recent sexual exposure or STI S/S:
    - Blood x HBsAg + anti-HCV + anti-HIV + VDRL
    - First-catch urine/high vaginal swab x gonorrhoea + chlamydia + trichomonas
    - +PO doxycycline ×1 week or PO azithromycin ×1 dose
    - Consult URO/GYN
  - PO pyridium for symptomatic relief (not for foley irritation)

- **Urine m'stix interpretation**:
  - **WBC +ve**: pyuria = inflammatory, non-specific
  - **Nitrite +ve**: bacteriuria, treat if symptomatic/pregnant/transplant kidney/recent urological procedure
  - **RBC +ve**: haematuria — workup for UTI, stones, CA, GN, rhabdomyolysis
  - **Sugar +ve**: glycosuria — FG + HbA1c + H'stix
  - **Ketone +ve**: workup for DKA if H'stix elevated or on SGLT2i
  - **Protein +ve**: ddx UTI/CKD/nephrotic syndrome — urine c/st + UPCR; if DM: UACR + ACEI if microalbuminuria
