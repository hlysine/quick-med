# General Drain Management

## Types of drains you will see in ward
* **Surgical drains**
    * Post-op drains: Round drain (R/D), Redivac drain
    * Pigtails for draining abscess/ collection/ ascites
    * PTC/ PTBD
    * PCN
* **Chest drain / pleural pigtail**
* **Foley / Paul's tube**

## Tasks you might need to do
* **Aspirate drain** (document amount and content) / **flush drain** (with 5ml NS, document any resistance)
* **Shift drain** (when low output, may try shifting out before off drain)
    * **Remove the drain if side holes are seen** (for chest drain, be highly alert and act quick!)
    * **Do not shift PCN/PTC/ PTBD** (supposed to be in fixed position)!
* **Off drain** (usually when nil output despite shift/ flush/ aspirate for a few days or when drain dislodged)
* **Re-anchor drain** (when stitch loosens - rmb to flush/aspirate +/- XR to confirm no dislodgement first!)

> **Before off/shift drain, check:**
> 1. **The exact management order** written by MO (especially if there are more than one drain)
> 2. **OT record:** sometimes will tell which drain exit site corresponds to which drain tip location, and whether the tip was cut before placement
> 3. **Correct patient identity**
> 4. **Correct drain** (DO NOT trust the bedside bag label)

> **After off drain,** check carefully and document if the **tip is intact!** If tip edge is irregular, try to find out if the tip is cut during intra-op placement, if not then order XR to look for retained tip!

> **Some pigtails are locking pigtails** - you must **cut the tension line** before the end loop can uncoil! Do not pull it out with brute force (type of pigtail inserted can be found in procedure record)

## Is suturing needed for off/shift drain? What else should we prepare?
* **Shift drain:** most of the time, can 捽 it out with slow rotating motion without cutting anchoring stitch
* **Off drain:** most of the time no need to suture the wound, just apply Opsite then pressure dressing
* **Off chest drain:** (document every step)
    * **Apply purse string suture first** (no need for pleural pigtail - just Opsite + pressure dressing)
    * **Off drain under Valsalva/ end inspiration/ end expiration** (/expiratory phase if cannot cooperate)
    * **Immediately tighten purse string suture** once drain removed, and tie it under tension
    * **Document SpO2** and any discomfort, **order obs Q1H x4 and post-CXR** to exclude PTX
    * Wash till light haematuria only or clear; if not clear, then order bladder irrigation
* **Shift chest drain: prepare standby materials** for removing chest drain; remove swiftly when see side hole!

## Common Problems Encountered

### Low PCN/ PTC/ PTBD output
* **Concept:** don't forget our body has **internal drainage systems** - external drainage is just extra help!
    * If patient can void and total urine output is normal, then low PTC output can be normal
    * Likewise, low PTC/ PTBD output can be a good sign that biliary obstruction is resolving
    * If confident, URO/SUR MOs may even order to spigot the drain to test if internal drainage is adequate (if yes, can off the drain!) - **but if patient develops S/S of urosepsis/biliary sepsis while spigotted, order reconnect to bedside bag immediately!**
* Check for any visible blockage/ kinking/ dislodgement
* Flush with 5ml NS and then aspirate
    * Unable to flush = blocked/ kinked/ dislodged
    * Flush smooth, unable to aspirate = drain tip displaced to somewhere in peritoneum (your flushed volume went to somewhere deep in the peritoneum)
    * Flush smooth, aspirate flushed volume then meet resistance: patent (there isn't any urine/bile left for you to aspirate)
* **KUB/AXR** to check if drain is in situ
* **Bld x CBC d/c + RFT** (*for PCN*) or **LFT** (*for PTC/ PTBD*)
* **If any S/S of urosepsis/biliary sepsis: urgent consult URO/SUR** -> need **urgent revision**

### High drain output
* Check I/O balance
* If surgical drains - make sure it is not bleeding / abnormal
    * If there is sudden +/- rapid increase in drain output especially early post-op, check vitals (**tachycardia/ decreased BP**), CBC - **may need EOT if rebleeding!**
    * For post cholecystectomy drains - **bile stained colour drains are abnormal!** (need to rule out bile leak/ stump leak) - monitor vitals, LFT (might need confirmation by CT scan/ ERCP)
* Depending on the drain output, might cause **electrolyte imbalance** -> monitor and treat accordingly
    * E.g. high bile output in PTBD may lead to hyponatraemia (replace with NS)

### Dislodged drain
* **Check if the drain is really dislodged (e.g. any output, whether side holes are seen)**
    * Might need to confirm with XR if not sure
    * Depending on the type of drains - if PCN/ PTC/ PTBD dislodged, need to monitor vitals (any s/s of sepsis) and RFT or LFT to see if reinsertion is necessary
    * Otherwise for surgical drains usually not a big deal if dislodged, just remove is ok
* **If side holes seen** - the drain is not useful anymore, just remove it
    * If it is a **chest drain dislodgement, order SpO2 monitoring and urgent CXR for PTX!**
* If just anchoring stitch loosened but drain is in situ, re-anchor with suture x 1

### Self-pulled-out (part or whole of the) drain
* **Remove the remaining parts of the drain**
* Check if the **drain tip is intact;** if edge is irregular, **order XR for retained fragment!**
