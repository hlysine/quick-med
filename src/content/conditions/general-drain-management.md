- **Types of drains**: Surgical drains (post-op round drain/Redivac, pigtails for abscess/collection/ascites, PTC/PTBD, PCN); Chest drain/pleural pigtail; Foley/Paul's tube

- **Tasks**: aspirate drain (document amount + content); flush drain (5ml NS, document resistance); shift drain (when low output; remove if side holes seen – for chest drain be alert!; do NOT shift PCN/PTC/PTBD); off drain (usually nil output despite shift/flush/aspirate, or dislodged); re-anchor drain (when stitch loosens – flush/aspirate ±XR first)

- **Before off/shift drain**: check exact management order written by MO; check OT record; correct patient identity; correct drain (do NOT trust bedside bag label); after off drain: check if tip intact; if edge irregular, order XR for retained fragment; **some pigtails are locking pigtails** – cut tension line before uncoiling (do not pull with brute force)

- **Off chest drain**: purse string suture first (no need for pleural pigtail – just Opsite + pressure dressing); off drain under Valsalva/end inspiration/end expiration; immediately tighten purse string; document SpO2 + discomfort; obs Q1H ×4; post-CXR to exclude PTX

- **Common problems encountered**:
  - **Low PCN/PTC/PTBD output**: if patient can void normally and total urine output is normal, low PCN output may be normal; check for blockage/kinking/dislodgement; flush with 5ml NS then aspirate; KUB/AXR; blood x CBC d/c + RFT (for PCN) or LFT (for PTC/PTBD); if S/S sepsis: urgent consult URO/SUR
  - **High drain output**: check I/O balance; if surgical drains: ensure not bleeding/abnormal (sudden increase early post-op: check vitals/CBC, may need EOT if rebleeding); post-cholecystectomy: bile-stained = abnormal (rule out bile leak – monitor LFT, may need CT/ERCP); depending on output, may cause electrolyte imbalance
  - **Dislodged drain**: check if really dislodged (output, side holes visible); confirm XR if unsure; PCN/PTC/PTBD dislodgement: monitor vitals + RFT/LFT; surgical drains: usually not a big deal; if side holes seen: remove; if anchoring stitch loosened but in-situ: re-anchor
  - **Self-pulled-out drain**: remove remaining parts; check tip intact; if edge irregular: XR for retained fragment
