## Symptoms

- Bloating / belching
- Acid reflux
- Heartburn

## Causes

- GERD (± Barrett's oesophagus)
- Gastritis / PUD (± H. pylori)
- Gastric / pancreatic CA / HCC
- Pancreatitis
- Mesenteric ischaemia
- Functional dyspepsia

## Assessment

- Any epigastric pain / GI bleed symptoms?
- Abdomen: soft, epigastric tenderness
- Previous OGD findings and treatment (H. pylori eradication confirmed?)

## Management

### Symptomatic Relief

| Symptom | Drug |
|---|---|
| Acid reflux | Triact (antacid) / Pepcidine (H2 antagonist) / Pantoloc (PPI) |
| Belching / bloating | Gasteel / Gaviscon |

- Refer to SUR if persistent
- If epigastric pain/GI bleed: refer to **Abdominal Pain / Haematemesis / PR Bleed sections**

### H. Pylori Eradication (Triple Therapy ×2 weeks)

- **PO Pantoloc 40mg BD** or Nexium 40mg BD
- **PO Amoxicillin 1000mg BD** (if penicillin allergy: PO Flagyl 500mg TDS)
- **PO Clarithromycin 500mg BD**
- Withhold statin while on clarithromycin (rhabdomyolysis risk)
- Avoid Nexium with Plavix (use Pantoloc/lansoprazole instead)

### Eradication Test (must arrange; ~80% success rate)

| Ulcer Type | Test |
|---|---|
| Gastric ulcer (GU) | Repeat OGD with biopsy |
| Duodenal ulcer (DU) | Urea breath test / stool antigen test |

- Timing: >4 weeks after antibiotics
- Withhold PPI 2 weeks before test
- Do not test during acute UGIB episode
