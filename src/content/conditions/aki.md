## Definition

- Serum creatinine rise to **>1.5× baseline** within 1 week, **OR**
- Urine output **<0.5 ml/kg/hr for 6 hrs**

---

## Causes

| Category | Examples |
|---|---|
| **Pre-renal** | Dehydration, shock, cardiorenal syndrome, hepatorenal syndrome, bilateral RAS + ACEI/ARB/NSAID |
| **Renal** | GN, AIN, ATN (UTI, nephrotoxic drugs: NSAIDs / aminoglycosides / acyclovir / tenofovir / radiocontrast; tumour lysis / multiple myeloma / rhabdomyolysis) |
| **Post-renal** | Obstructive: stone / tumour / stricture |

---

## Complications — AEIOU

| Letter | Complication |
|---|---|
| **A** | Acidosis |
| **E** | Electrolyte abnormalities (hyperK / hyperPO4 / hypoCa) |
| **I** | Intoxication / drug toxicities |
| **O** | Oedema / fluid overload |
| **U** | Uraemic pericarditis / encephalopathy (pericardial effusion / seizures) |

---

## Assessment

- Vitals, fluid status
- Urinary symptoms: loin pain / dysuria / haematuria / turbid urine / LUTS
- Review drugs: nephrotoxics, anticholinergics / opioids for AROU, renally-excreted drugs
- Creatinine baseline and trend + urea and electrolytes

---

## Management

### Monitoring
- **Chart U/O**; if severe AKI: insert Foley for accurate monitoring

### Initial Workup
- Blood: **CBC d/c + RFT**
- **MSU / CSU**: m'stix + c/st
- **PVRU x1**
- If obstruction suspected: **KUB + urgent USG urinary system** / NCCT A+P stone protocol

### Fluid Management
- If **dry**: IV fluid replacement + maintenance; encourage oral fluid intake
- If **fluid overloaded**: fluid restriction <1.2 L/day; IV Lasix 40 mg Q12H
  - Renal failure: increase dose
  - Heart failure: increase frequency
  - Nephrotic: increase dose + frequency + add IV albumin 20 g/day

### Drug Management
- If septic / UTI / elevated WCC: empirical antibiotics (per past urine c/st)
- **Off all nephrotoxic drugs** (NSAIDs / aminoglycosides / acyclovir / tenofovir)
- **Adjust or stop** all renally-excreted medications based on latest CrCl

### Treat Complications
- Address AEIOU; consult renal / ICU for RRT if refractory

### If No Improvement
- Monitor: CBC d/c + L/RFT + Ca/PO4 + VBG
- Consider: USG / NCCT / consult urology for obstructive cause
- Consider consult MED for workup of intrinsic renal disease (leave these investigations for MO):
  - ANCA, ANA, anti-dsDNA, anti-GBM, C3/C4, HBsAg, anti-HCV, SPE, Ig pattern, FLC
  - Urine protein electrophoresis (UPE)

---

## Contrast CT in CKD (eGFR <45, Not on Dialysis)

- Use **plain CT / USG** if adequate for clinical question
- If proceeding with contrast: **volume expansion with NS**
- NaHCO3 / acetylcysteine: alternative but no evidence of benefit
