- Gross (red/pink urine) or microscopic (urine m'stix RBC +ve)

- **Causes**: recent instrumentation (foley/FC), UTI, urinary stones, urinary tract malignancy, bleeding BPH, nephritic syndrome (microscopic + proteinuria), systemic (antiplatelets/anticoagulants/coagulopathy)

- **False positives**: food dye (beetroot)/drug staining (pyridium, nitrofurantoin, rifampicin, phenytoin), menstruation, myoglobinuria (rhabdomyolysis)

- **Complications**: clot retention (AROU/blocked foley), anaemia

- **Assessment**: severity (vitals/Hb drop/blood clots); fever; urinary symptoms (stone symptoms, clot retention/LUTS/AROU); abdominal exam, costovertebral tenderness; PR exam (prostate); background (urological disease/workup, antiplatelet/anticoagulant)

- **Management — gross haematuria**:
  - If only mild after foley insertion: keep obs; chart I/O + haematuria
  - Blood x CBC d/c + RFT + clotting; MSU/CSU x m'stix + c/st + cytology; KUB; PVRU
  - If PVRU >350ml or blood clots: **3-way haematuria catheter (3-way 24Fr foley)** to BSB; bladder washout ±bladder irrigation full rate
  - W/H antiplatelets (consult if DAPT), W/H ±reverse anticoagulation
  - Consult URO

- **Management — microscopic haematuria**:
  - Chart I/O + haematuria
  - Blood x CBC d/c + RFT + clotting
  - MSU/CSU x m'stix + c/st + microscopy (dysmorphic RBC, casts)
  - Consider consult/refer MED if not explained by UTI

- **DO NOT give Transamin for haematuria** (precipitates clot retention)
