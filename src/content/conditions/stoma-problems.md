## Stoma Bleeding

- **Most important:** rule out GI bleeding (treat as PR bleed if melaena/fresh blood from stoma)
- **Most common cause:** peristomal bleeding (granulation tissue/friction/skin wounds)
- Management: **direct pressure**
  - If not stopping: apply **adrenaline gauze**

## High Stoma Output (>1L/day)

> Requires aggressive fluid and electrolyte replacement (especially **magnesium**)

- Ideally replace 400–800ml/day
- **Strict I/O chart**
- Rehydration aiming U/O **0.5ml/kg/day**
- Monitor RFT + Ca/PO4 + Mg + VBG

### Anti-motility Agents

> Avoid if high R/T output (i.e. still in ileus)

- PO Imodium 2mg TDS
- Codeine

### Anti-secretory Agents

- PPI
- Octreotide

## Stoma Prolapse

- Check for **ischaemia/gangrene** (healthy mucosa?)
- Check for **positive output** (exclude IO)
- Try **manual reduction** (± IM tramadol before reduction)
- Can try **D50 gauze** to reduce swelling before reduction
