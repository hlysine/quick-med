## Causes

| Level | Examples |
|---|---|
| **Brain** | Head injury, CVA, SOL, encephalitis, demyelination, seizure / post-ictal |
| **Spinal cord** | Trauma, CVA, myelopathy, transverse myelitis, B12 deficiency |
| **Nerve root** | Cauda equina syndrome, radiculopathy |
| **Neuropathy** | Hyperventilation, hypocalcaemia, DM, alcoholic, B1/B6/B12 deficiency, hypothyroid, CKD, CLD, drugs, autoimmune, paraneoplastic, multiple myeloma, lymphoma, HIV, Lyme disease, GBS, CIDP, CMT, nerve entrapment/trauma |

## Assessment

- Onset and trigger (e.g. cyclist's palsy, Saturday night palsy)
- **Pattern**:
  - **Perioral** → local anaesthetic systemic toxicity
  - **Face** → trigeminal distribution
  - **Sensory level** → spinal cord
  - **Dermatomal** → nerve root
  - **Single peripheral nerve territory** → mononeuropathy
  - **Glove and stocking** → polyneuropathy
- Associated symptoms: weakness, headache, dysarthria, sphincter disturbance
- **Full neurological exam**: cranial nerves, 4-limb power/reflexes/sensation, cerebellar signs
- **If cord suspected**: spinal tenderness, SLR, PR

## Management

- **CNS origin**: brain/cord imaging + neuro obs
- Bloods: FG + HbA1c ± TFT + B12
- **Neurobion** 1 tab daily
- **Neuropathic pain**: Lyrica (pregabalin) / gabapentin
- Nerve conduction studies (NCS)
- **Consult / refer neurology** if unexplained
