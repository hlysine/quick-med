Can be focal or generalised, convulsive or non-convulsive.

**Presentation**: focal/generalised twitching, up-rolling eyeballs, tongue biting, double incontinence; impaired consciousness; post-ictal drowsiness/confusion/headache/Todd's paresis.

## Causes

- Epilepsy
- CNS pathology: HI, stroke, infection, tumour
- Metabolic: hypoglycaemia, electrolyte disturbances
- Drug intoxication/withdrawal

## Complications

- Fall / HI
- Rhabdomyolysis

## Assessment

- Vitals — if **fever + seizure**: manage as CNS infection → blood c/st + LP
- **GCS + H'stix stat**
- History: trigger, witness, LOC/fall/HI, current symptoms, drug history
- Full neurological exam: CN, 4-limb power, cerebellar signs, meningism

## Management

### Airway First

- If **GCS <8**: call MO for intubation (unless DNACPR not for intubation)
- **Ensure airway protection BEFORE administering benzodiazepines**

### Benzodiazepines

Prescribed "as directed prn if seizure >30 sec" (if ongoing: + stat):

| Route | Agent | Dose |
|---|---|---|
| **IV** (preferred) | Lorazepam (Ativan) | 4 mg bolus; repeat once after 10–15 min if not aborted |
| **IV** (alternative) | Diazepam (Valium) | 5–10 mg bolus (action <20 min, 50% recurrence within 2 hrs — must add long-acting agent) |
| **If no IV access** | Midazolam | 10 mg IM / buccal / intranasal |
| **If no IV access** | Diazepam | 0.2 mg/kg PR up to 20 mg |

### Long-Acting Anti-Epileptic

- If **not already on a long-acting anti-epileptic**: add one even if convulsions have ceased (typically **IV Dilantin/phenytoin**)

### Ongoing Monitoring and Investigations

- Neuro obs Q1H
- H'stix monitoring
- Urgent CT brain
- **Bloods**: CBC d/c + L/RFT + Ca/PO4 + Mg + RG + NH3 + clotting + serum toxicology; ± ABG; + blood c/st if fever
- Urine toxicology
- ± LP, EEG
- Consult neurology

### Non-Convulsive Status Epilepticus (NCSE)

- If **GCS does not return to baseline within 30–60 min** after convulsions subside: suspect NCSE → **urgent EEG**
