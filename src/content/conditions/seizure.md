# Seizure

> **Can be focal or generalised, convulsive or non-convulsive (just impaired consciousness)!**

## Presentation
* Focal/generalised muscle twitching (e.g. facial/periorbital twitching / rhythmic nystagmoid eye jerks), up-rolling eyeballs, tongue biting, double incontinence
* Impaired consciousness, post-ictal drowsiness/confusion/headache / Todd's paresis

## Causes
* Epilepsy
* CNS pathology: HI, stroke, infection, tumour
* Metabolic: hypoglycaemia, electrolyte disturbances
* Drug intoxication/withdrawal

## Complications
* Fall/HI
* Rhabdomyolysis

## Assessment
* Vitals (*if fever + seizure, manage as CNS infection, need bld c/st & LP – refer to section on ↓ GCS*)
* **GCS, H'stix stat**
* History: trigger, witness, **LOC/fall/HI**, current symptoms, **drug history**
* **Full neuro exam:** CN (pupils, EOM, CN7, dysarthria), 4 limbs power, cerebellar signs, **meningism**

> If GCS < 8, first thing is airway protection; must be ensured before giving benzodiazepine!

> If GCS does not return to baseline in 30-60min after convulsions subside, must suspect non-convulsive status epilepticus (NCSE) and order urgent EEG

## Management
* If GCS < 8: call MO x **intubation** for airway protection (unless DNACPR not for intubation)
* **Prescribe benzodiazepine** "as directed prn if seizure >30 sec" (if seizure ongoing: + stat)
    * **IV lorazepam (Ativan) 4mg bolus**, may repeat once after 10-15min if not aborted
    * **Or IV diazepam (Valium) 5-10mg bolus** (*duration of action <20min, 50% recurrence within 2hrs, so must add long-acting anti-seizure drug*)
    * If no IV access: IM/ buccal/ intranasal midazolam 10mg, or rectal diazepam gel 0.2mg/kg up to 20mg
* **If not on long-acting anti-epileptic**, add one even if convulsions ceased, typically Dilantin (see table)
