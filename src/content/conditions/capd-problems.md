- **Cloudy effluent** (treat as CAPD peritonitis):
  - **Diagnostic criteria**: clinical features (fever/abd pain/cloudy effluent); WCC ≥100 cells/mm³ + neutrophils >50%; +ve CAPD dialysate culture
  - **Assessment**: save CAPD effluent for inspection; septic workup; send CAPD fluid for gram stain/culture/microscopy/cell count
  - **Management**:
    - Rapid flushing PDF 2L ×3 + IP heparin 500-1000 units/L
    - Increase CAPD regimen to 4 bags/day + IP heparin
    - If no septicaemia/fever: IP cefazolin 1g daily + IP ceftazidime 1g daily + stat
    - If septicaemia/fever: IV cefazolin + IV cefepime (gentamicin if anuric) + PO nystatin
    - Chart I/O + BW + monitor RFT; repeat PD fluid x WCC + gram smear + culture; check clinical response
    - If limited response within 1-2 days: consider other causes (perforated viscus, appendicitis etc.)

- **Exit site infection**: purulent discharge; erythema ±indurations/fluctuance around catheter insertion site; wound swab for microscopy/culture; antibiotics per swab results (topical gentamicin cream, oral abx e.g. cloxacillin 500mg QID ×2 weeks); consult renal

- **Low Tenckhoff output**: causes = blockage by fibrin/kinking (external or faecal-loaded bowel)/low osmotic pull due to hyperglycaemia
  - Management: H'stix monitoring → Actrapid if high; KUB (check kinking) → if constipation: laxatives/mobilise/change posture; if blockage: consult renal for flushing/IP heparin/IP urokinase/Tenckhoff revision
